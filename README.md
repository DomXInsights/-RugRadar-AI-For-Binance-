# -RugRadar-AI-For-Binance-
RugRadar AI is an on-chain analysis assistant that detects potential team dumping before retail investors notice. Using Binance Web3 data, it audits contracts, analyzes creator wallets, tracks smart-money signals, and monitors social trends.The system combines these signals into a single risk score to warn trader when insider exits may be happening
## Project Structure

- `docs/pipeline.md` — explains the 5-step RugRadar analysis flow
- `docs/architecture.md` — explains how OpenClaw, Binance Web3 data, and RugRadar connect
- `docs/skills.md` — describes the five analysis skills
- `api/` — contains the core RugRadar skill modules and engine
- `demo/example-report.json` — sample RugRadar output report
